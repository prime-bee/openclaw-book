# openclaw-book

Um livro didático em HTML sobre o **OpenClaw**, organizado para leitura local no navegador.

## O que é

Este projeto transforma a documentação oficial do OpenClaw em um formato mais pedagógico, com:

- capítulos navegáveis em HTML
- referências para a documentação oficial
- receitas passo a passo
- troubleshooting
- bibliografia operacional

A proposta é servir como uma ponte entre:

- a documentação oficial do OpenClaw
- o aprendizado prático de quem está começando
- e um formato de leitura mais agradável para estudo contínuo

## Estrutura

```text
book/
  index.html
  styles.css
  assets/
  chapters/
  README.md
```

### Arquivos principais

- `index.html` — página inicial e sumário
- `styles.css` — estilos do livro
- `chapters/` — capítulos em HTML
- `assets/` — imagens e recursos visuais

## Como abrir

Não precisa de build nem servidor.

Basta abrir:

- `index.html`

em qualquer navegador moderno.

## Conteúdo atual

O livro cobre, entre outros temas:

- visão geral do OpenClaw
- arquitetura
- gateway
- CLI
- memória e workspace
- canais
- modelos e provedores
- multi-agent
- automação com cron e heartbeat
- sandbox e segurança
- browser automation
- receitas passo a passo
- troubleshooting
- referências oficiais

## Objetivo do projeto

Este projeto não tenta substituir a documentação oficial.

O objetivo é:

1. reorganizar o conteúdo de forma didática
2. facilitar a navegação e estudo
3. conectar conceitos com operação prática
4. oferecer um material mais próximo de um livro técnico

## Fontes

A principal referência é a documentação oficial do OpenClaw:

- https://docs.openclaw.ai
- https://github.com/openclaw/openclaw

Este projeto também foi desenvolvido com base na documentação local instalada no ambiente do autor.
