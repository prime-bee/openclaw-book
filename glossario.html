Glossário essencial

Termo
Explicação
Gateway
O processo central do OpenClaw que recebe mensagens, roteia sessões, integra canais e expõe controle.
Session
Unidade de contexto conversacional. É o “fio” de memória de curto prazo de uma conversa.
Workspace
Pasta operacional do agente: arquivos, memória, docs, identidade e materiais auxiliares.
MEMORY.md
Memória longa e curada do agente.
Heartbeat
Turno periódico na sessão principal para checagens contextuais.
Cron
Agendador preciso para jobs recorrentes ou one-shot.
Binding
Regra que decide qual agente recebe qual mensagem.
AgentDir
Diretório de estado do agente: auth, models registry e metadados operacionais.
Sandbox
Execução isolada de ferramentas via container, para reduzir risco.
ACP
Harness/ambiente específico para sessões persistentes de agentes externos como Codex/Claude Code.
CDP
Chrome DevTools Protocol, usado no controle de browser.
