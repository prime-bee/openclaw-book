A CLI do OpenClaw

A CLI é ampla. Em vez de decorar tudo, vale aprender os grupos principais.

Setup

setup, onboard, configure

Operação

status, health, logs, gateway

Chat e agentes

message, agent, agents, sessions

Ambiente

channels, models, memory, skills

Comandos para decorar primeiro

openclaw status
openclaw gateway status
openclaw channels status --probe
openclaw models status
openclaw logs --follow

Exemplos práticos

1) Ver canais

openclaw channels list
openclaw channels status --probe

2) Rodar uma mensagem de agente

openclaw agent --message "Resuma meus arquivos de memória"

3) Ver memória indexada

openclaw memory status
openclaw memory search "preferências do usuário"

Dica didática

Use a CLI como se fosse uma árvore:

openclaw <área> <ação>

Exemplos:
openclaw gateway restart
openclaw models list
openclaw channels add

Exercício

Prática: escolha três comandos que você provavelmente usaria toda semana e escreva por que cada um deles importa.
