# Security policy

## Escopo

Este repositório publica um site estático e documentação editorial sobre OpenClaw. Vulnerabilidades relevantes aqui tendem a envolver:

- conteúdo injetável em HTML/JS do próprio site
- links inseguros ou regressões que exponham navegação a risco
- publicação incorreta de arquivos que não deveriam ir para o Pages

## Como reportar

Abra uma issue privada se o repositório suportar esse fluxo; caso contrário, abra uma issue pública somente para problemas seguros de divulgar. Não inclua segredos, tokens, credenciais nem detalhes do seu ambiente local.

## O que esperamos em uma boa report

- página afetada
- impacto percebido
- passos de reprodução
- sugestão de correção, se houver
